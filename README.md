# 엔카 중고차 품질 점수 Chrome Extension

엔카(`car.encar.com`) 중고차 목록 페이지에서 각 차량의 품질을 자동으로 **100점 만점으로 점수화**하여 배지 형태로 표시해주는 Chrome 확장 프로그램입니다.

## 주요 기능

- 차량 목록에 **점수 배지** 자동 표시 (S/A/B/C/D 등급)
- 배지 호버 시 항목별 점수 **툴팁** 표시
- **최소 점수 필터**: 기준 점수 미만 차량 숨김 처리
- **가중치 커스터마이징**: 팝업에서 각 항목별 가중치 슬라이더 조절

## 점수 항목 (기본 가중치 합계: 100점)

| 항목 | 기본 점수 | 설명 |
|------|-----------|------|
| 사고/보험이력 | 20점 | 보험금 규모(신차가 대비 비율)로 판단. 조회불가·비공개 시 0점 |
| 가격 | 25점 | 연식별 기대 감가율 대비 실제 가격 비교 |
| 렌트이력 | 20점 | 렌트/영업용 이력 여부 |
| 성능점검 | 15점 | 엔카진단(프레임/패널 교환) 또는 일반 성능점검표 기반 |
| 주행거리 | 10점 | 연간 평균 주행거리 기준 |
| 소유주이력 | 10점 | 소유주 변경 횟수 (1인 차주 만점) |

## 설치 방법

1. 이 저장소를 클론 또는 다운로드
2. Chrome에서 `chrome://extensions` 접속
3. **개발자 모드** 활성화
4. **압축 해제된 확장 프로그램 로드** → 이 폴더 선택

## 사용 방법

1. `https://car.encar.com/list/...` 페이지 접속
2. 차량 카드에 자동으로 분석 배지가 표시됨
3. 배지 **호버** → 항목별 점수 상세 확인
4. 우측 하단의 **점수 오버레이** 또는 **팝업** → 필터·가중치 조절

## 파일 구조

```
├── manifest.json       # 확장 프로그램 설정
├── constants.js        # 기본 가중치 상수
├── scoring.js          # 점수 계산 로직
├── detail-parser.js    # 엔카 API 데이터 파싱
├── content.js          # 페이지 주입 스크립트 (배지·필터)
├── popup.html/js/css   # 팝업 UI
└── styles.css          # 배지·툴팁 스타일
```

## 사용 API

| API | 용도 |
|-----|------|
| `/v1/readside/vehicle/{id}` | 차량 기본 정보, 신차가, 옵션 목록 |
| `/v1/readside/vehicles/car/{id}/options/choice` | 선택 옵션 가격 합산(신차가 보정) |
| `/v1/readside/record/vehicle/{id}/open` | 보험·사고이력, 소유주 변경이력 |
| `/v1/readside/inspection/vehicle/{id}` | 성능점검 기록 |
| `/v1/readside/diagnosis/vehicle/{id}` | 엔카진단 결과 |
