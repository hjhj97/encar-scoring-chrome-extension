## 기능 추가 이력

### 사고/보험이력 점수 통합
- `insurance`(보험이력)와 `accident`(사고이력)를 단일 카테고리로 통합
- 보험금 건수 기반 → **보험금 총액 / 신차가 비율** 기반 점수로 변경
  - `< 10%`: 경미 (75%)
  - `10~20%`: 중간 (45%)
  - `> 20%`: 심각 (10%)
  - 조회불가 / 비공개: 0점

### 소유주 변경이력 점수 추가
- 소유주 변경 횟수에 따른 감점 (0회 만점, 1회 70%, 2회 40%, 3회+ 10%)
- 툴팁 및 팝업 슬라이더에 항목 추가

### 신차가 계산 개선 (선택 옵션 포함)
- 기존: `category.originPrice` (기본 트림가)
- 개선: `/options/choice` API로 선택 옵션 가격을 합산하여 실제 신차가 계산
- 감가율 기반 가격 점수에 적용

### 가격 점수 로직 개선
- 단순 평균 비교 → **연식별 기대 잔존가치율** (`RETENTION_BY_AGE`) 대비 실제 가격 비교
- 기대가 대비 저렴할수록 높은 점수, 비쌀수록 낮은 점수

### 엔카진단 성능점검 로직 추가
- 엔카진단(`diagnosisCar: true`) 차량: 프레임 교환 여부, 외부패널 교환 여부 별도 판정
- 프레임 교환: `-12점` (심각), 외부패널 교환: `-3점` (경미)

### 점수 오버레이 위치 변경
- 페이지 내 점수 오버레이를 우측 하단으로 이동

### 보험이력 비공개 차량 감점
- `condition.accident.recordView === false` 차량: 사고/보험이력 0점 처리
